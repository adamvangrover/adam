# Getting Started with Adam Platform

This tutorial will guide you through setting up the Adam Platform and running your first autonomous agent.

## Prerequisites

1.  Python 3.12+ installed.
2.  `uv` installed for dependency management.

## Installation

Clone the repository and sync dependencies:

```bash
git clone https://github.com/adamvangrover/adam.git
cd adam
uv sync
```

## Running the First Agent

Execute the Interactive CLI to launch the meta-orchestrator:

```bash
uv run python scripts/run_adam.py
```
