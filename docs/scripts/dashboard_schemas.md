# Dashboard Data Schemas

HTML Dashboards generated by scripts (like `generate_daily_index.py`) often rely on embedded JavaScript arrays to render dynamic components like masonry grids and gated modals.

## The `modules` Array Schema
The standard array format expected by the frontend parsing logic is `const modules = [...]`.

```javascript
const modules = [
    {
        title: "Credit Architect Protocol", // Display Title
        category: "credit",                 // Category tag for filtering
        description: "Deep dive...",        // Card summary text
        link: "credit_protocol.html",       // Href for the card link
        date: "2026.03.15",                 // Display date
        locked: true                        // Optional: Marks content as gated/premium
    },
    // ...
];
```

## Gated Content Mechanism
When the `locked` property is `true`, the UI will apply the `.gated-content-blur` class. The `#authModal` script requires a valid key (e.g., 'admin', 'adam', 'genesis') to remove this blur state and reveal the underlying link.
