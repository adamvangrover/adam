# Adam v23.5 Showcase Updates

## Overview
The "Mission Control" interface has been significantly upgraded to reflect the v23.5 "Cyber-Financial" aesthetic and functionality.

## Changes Implemented

### 1. Unified Aesthetic
- **CSS Architecture**: Centralized styles in `css/style.css` with utility classes for `.glass-panel`, `.neon-text`, and `.scan-line` animations.
- **Theme**: Enforced Slate-900 dark mode with Cyan/Emerald accents.

### 2. Data Layer (`mock_data.js`)
- Expanded `window.MOCK_DATA` to include:
    - `agents`: Populated with active/idle states for Orchestrators and Sub-Agents.
    - `reasoning_circuits`: Node/Edge data for the Neural Dashboard graph.
    - `entity_graph`: Knowledge graph data for the Financial Twin.
    - `deep_dive_metrics`: Valuation and Risk data for reports.

### 3. Interactive Modules

#### Neural Dashboard (`neural_dashboard.html`)
- **Engine**: Integrated `vis-network` for force-directed graph visualization.
- **Features**:
    - Renders "Reasoning Circuits" (SNC, Sentiment, Red Team).
    - Live "Thought Stream" log with color-coded events.
    - Pulse animations for active nodes.

#### Financial Twin (`financial_twin.html`)
- **Engine**: Integrated `vis-network` for entity mapping.
- **Features**:
    - Interactive Knowledge Graph showing Suppliers, Competitors, and Targets.
    - "Contagion Simulation" toggles (Supply Shock, Credit Default) that animate risk propagation (red edges).
    - Slide-out "Real-Time Telemetry" panel.

#### Deep Dive Explorer (`deep_dive.html`)
- **UI**: Converted text cards into interactive widgets.
- **Features**:
    - Valuation Bar Chart (Intrinsic vs Market).
    - Risk Probability Gauge (SVG-based).
    - Print/PDF export mode.

#### Agent Registry (`agents.html`)
- **UI**: Grid view with status indicators.
- **Features**:
    - Complex filtering (Type, Status).
    - Simulated "Terminal Console" showing startup logs for selected agents.

#### The Vault (`data.html`)
- **UI**: VS Code-style file explorer.
- **Features**:
    - Recursive Tree View.
    - Syntax-highlighted preview using `Prism.js`.

### 4. Navigation
- Updated `index.html` to serve as a central hub with links to all new modules.
- Added placeholders for "Platform Trading", "Robo Advisor", and "Research" as requested.

## Usage
Open `showcase/index.html` in a browser (served via a local server) to explore the updated interface.
