# Adam Mission Control (Showcase)

The `showcase/` directory contains the visual interfaces for interacting with Adam v26.0. These are static HTML/JS dashboards that connect to the backend API.

## üñ•Ô∏è Key Dashboards

### 1. Meta-OS Command Center (`index.html`)
The primary gateway to the Adam ecosystem.
*   **Role:** Central Hub / Desktop Metaphor.
*   **Features:** Window management, taskbar, and quick access to all other apps.

### 2. Office Nexus (`office_nexus.html`)
A productivity suite integration.
*   **Role:** Document & Workflow Management.
*   **Features:** Dynamic shortcut generation based on the filesystem manifest, theme customization.

### 3. Neural Dashboard (`neural_dashboard.html`)
The "System 2" visualizer.
*   **Role:** Real-time reasoning display.
*   **Features:** Stream of agent thoughts ("Intercom"), active system status, and "Brain" activity monitoring.

### 4. Crisis Simulator (`crisis_simulator.html`)
A "War Room" interface for running macro-economic stress tests.
*   **Capabilities:**
    *   Inject shocks (e.g., "Liquidity Freeze", "Suez Blockage").
    *   Visualize impact on LCR (Liquidity Coverage Ratio) and CET1 Capital.
    *   Real-time "Risk Radar".

### 5. War Room V2 (`war_room_v2.html`)
A grid-based monitoring station for live market data and system health.
*   **Widgets:** Ticker tape, breaking news feed, active agent list, server latency.

### 6. Deep Dive Reports (`report_*.html`)
Static, high-fidelity renders of Adam's deep dive analyses.
*   **Example:** `report_great_bifurcation_2026.html` - A comprehensive strategic outlook generated by the system.
