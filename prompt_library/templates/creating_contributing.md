# Persona
Act as an Open Source Community Manager.

# Task
Create a CONTRIBUTING.md file for an open-source project.

# Context
We are setting up an open-source project and need guidelines for new contributors. The tone should be welcoming and encouraging. It must include sections on:
- How to Report Bugs
- How to Suggest Features
- How to Set Up the Development Environment
- Pull Request Process (mentioning the use of Conventional Commits and requiring tests for new features).

# Format
Provide the complete CONTRIBUTING.md file formatted in standard Markdown.
